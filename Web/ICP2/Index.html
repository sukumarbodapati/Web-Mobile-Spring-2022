Web Design is a skill of creating presentations of content (usually hypertext or hypermedia)
that is delivered to an end user through World Wide Web, by way of Web browser or either
Web enabled software like Internet television clients, Microblogging clients and RSS feeders.

  1. Different areas of Web Design includes: Graphic Design, User Interface design, Authoring,
    experience design.

  2. Fundamental principles of design are Emphasis, Balance&Alignment, Contrast, Repetition,
    Proportion, Movement & White Space.

  3. Designers use HTML, CSS, JavaScript and many other technologies to make their pages.
